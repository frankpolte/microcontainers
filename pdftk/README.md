PdfTk docker image (minidocks/pdftk)
====================================

[PdfTk](https://www.pdflabs.com/tools/pdftk-server/) is a simple tool for doing everyday things with PDF documents.

Usage
-----

```bash
docker run --rm -v `pwd`:/app -w /app minidocks/pdftk -h
```
